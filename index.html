Why WET VST?

A different approach to plugin development

Fully Open Source

Most free plugins are closed-source binaries. WET plugins give you the complete source code under a permissive license. Learn, modify, contribute.

Pure VST3 SDK

Built directly on the official Steinberg VST3 SDK. No JUCE, no third-party frameworks. Clean, minimal, and easy to understand.

Complete Documentation

Every plugin includes full build instructions using only free tools: Visual Studio Build Tools, CMake, and Git. No paid licenses needed.

Windows Focused

Deliberately targeting the Windows ecosystem where most bedroom producers work with DAWs like Reaper and FL Studio.

Support Development

WET plugins are free forever. If they help your music, consider buying me a coffee to support continued development.
